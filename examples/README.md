# Examples

Sample Flutter apps used as fixtures for shotgun's own development and as documentation by example.

Planned:

- `counter_app/` — the Phase 1 smoke test. Flutter's default counter, but with one shotgun scene wired up.
- `notes_app/` — Phase 2 demo. List + detail + form, multi-locale, multiple scenes.

Each example is a real, runnable Flutter project with `shotgun_runner` as a `dev_dependency` and a working `shotgun.yaml`.
