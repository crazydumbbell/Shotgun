# Presets

Each preset is a single YAML file describing how a raw screenshot becomes a finished store image.

Planned built-ins:

- `vivid_gradient.yaml` — bold gradient bg, large caption, tilted phone (default)
- `minimal.yaml` — white/light bg, semibold caption, subtle shadow
- `feature_callout.yaml` — arrow + circle highlight + short caption per feature
- `lifestyle.yaml` — composite over a hand/desk background photo

Files come in Phase 1 / Phase 2. See [../docs/ARCHITECTURE.md](../docs/ARCHITECTURE.md) for the preset schema.
